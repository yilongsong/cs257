AUTHORS: Yilong Song, Skyler Kessenich

DATA: US Senate election data from 1976 to 2020

FEATURES CURRENTLY WORKING:
- View candidate election history by searching candidate name (Search by Candidate Name)
- View election data (% votes received for each candidate) for a given state in a given year (Search by State & Year)

FEATURES NOT WORKING:
- View election data of all available years for a given state (Search by State)
- View election data of all states for a given year (Search by Year)
- Put candidate election history results in pages to reduce length for long searches
- Display more election data for given state in a given year
